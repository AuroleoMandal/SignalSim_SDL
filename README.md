## SignalSim_SDL

Calculates and plots various waves and their resultant.

## Configuration

Configure by changing config.h

## Compilation

Compile using g++
(Use Linux Subsystem if on Windows e.g. MinGW)

Navigate to the project directory

#### Linux
```bash
make
```

#### Windows
```bash
start runWindows.bat
```
